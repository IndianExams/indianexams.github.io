Vue 3 Composition API Cheatsheet – Chapter 3

Lifecycle Hooks

Commonly Used Lifecycle Hooks

Hook When it runs Example
onMounted After component is mounted (DOM available) Fetch API data, initialize libraries
onUnmounted Before component is destroyed Cleanup (remove event listeners, stop timers)
onUpdated After reactive state updates the DOM Debug DOM changes, sync non-Vue libraries
onBeforeMount Right before DOM is mounted Rarely used, setup last-minute data
onBeforeUnmount Right before component teardown Cleanup before destroy
onBeforeUpdate Before DOM re-render caused by reactive changes Debug, measure performance

Example – Fetch data on mount

<script setup>
import { ref, onMounted } from "vue"

const users = ref([])

onMounted(async () => {
  const res = await fetch("https://jsonplaceholder.typicode.com/users")
  users.value = await res.json()
})
</script>

<template>
  <ul>
    <li v-for="user in users" :key="user.id">{{ user.name }}</li>
  </ul>
</template>

Example – Cleanup on unmount

<script setup>
import { ref, onMounted, onUnmounted } from "vue"

const time = ref(0)
let timer

onMounted(() => {
  timer = setInterval(() => {
    time.value++
  }, 1000)
})

onUnmounted(() => {
  clearInterval(timer)
})
</script>

<template>
  <p>Timer: {{ time }}</p>
</template>

✅ Key Takeaways

  • Lifecycle hooks in Vue 3 are functions imported from vue.
  • Perfect for data fetching, DOM access, and cleanup logic.
  • Composition API hooks replace Options API (mounted, created, etc.) with a function-based style.
