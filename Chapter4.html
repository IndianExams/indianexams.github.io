Template Syntax & Directives

Vue templates use special directives (prefixed with v-) to bind data, handle events, and control rendering.

1. Text Interpolation

<p>{{ message }}</p>
  • Renders the value of message into the DOM.
  • Reactive: updates automatically when message changes.

2. Attribute Binding (v-bind)

<img v-bind:src="imageUrl" />
<!-- Shorthand -->
<img :src="imageUrl" />
  • Dynamically binds attributes or props.
  • Shorthand: : instead of v-bind:.

3. Event Handling (v-on)

<button v-on:click="increment">+</button>
<!-- Shorthand -->
<button @click="increment">+</button>
  • Binds event listeners.
  • Shorthand: @ instead of v-on:.

4. Conditional Rendering

<p v-if="isVisible">Visible</p>
<p v-else>Hidden</p>

<p v-show="alwaysVisible">I am always in DOM, but toggled by CSS</p>
  • v-if / v-else removes/creates elements.
  • v-show toggles display: none (better for frequent toggling).

5. List Rendering (v-for)

<ul>
  <li v-for="(item, index) in items" :key="index">
    {{ index }} - {{ item }}
  </li>
</ul>
  • Loops through arrays or objects.
  • Always provide a unique :key.

6. Two-way Binding (v-model)

<input v-model="text" />
<p>You typed: {{ text }}</p>
  • Syncs data between input and component state.

7. Class & Style Bindings

<div :class="{ active: isActive, 'text-danger': hasError }"></div>
<div :style="{ color: activeColor, fontSize: fontSize + 'px' }"></div>
  • Bind dynamic classes and inline styles.

8. Slot Usage (Basics)

<!-- Parent.vue -->
<MyCard>
  <template #header>
    <h1>Custom Header</h1>
  </template>
  <p>Content inside slot</p>
</MyCard>

<!-- MyCard.vue -->
<div class="card">
  <slot name="header"></slot>
  <slot></slot> <!-- default slot -->
</div>
  • Named and default slots allow content projection.

✅ Key Takeaways

  • This chapter focuses on template features you'll use in almost every Vue 3 project.
  • Directives (prefixed with v-) are Vue's special template attributes.
  • Most directives have shorthand syntax (: for binding, @ for events).
